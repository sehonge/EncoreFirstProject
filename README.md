# EncoreFirstProject

<br/> 
수업에서 진행한 미니웹프로젝트로 가상의 피자 브랜드를 설정하여 웹사이트를 만들어 본 프로젝트이다.
순수 Java와 Javascript만을 사용해서 프로젝트를 마무리했다.

<br/>

### 후기
순수 java와 JDBC를 사용해서 구현한것이 아쉬웠기때문에 Spring과 JPA를 사용해서 리펙토링을 해봐야겠다.

추후에 리펙토링작업을 시작하면 링크를 연동해두겠습니당.
